# VitalSynth
HIPAA-safe synthetic ECG/PPG API

## Environment Setup

Create a `.env` file in the project root with the following variables:

```
MODEL_AES_KEY=...
VITALSYNTH_API_KEY=...
VITALSYNTH_PEPPER=...
SECRET_KEY=...
```

These values are not tracked in version control; each developer should supply their own.
